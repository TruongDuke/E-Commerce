# 🛒 E-Commerce Website

Website thương mại điện tử bán sách, CD/LP và DVD với tích hợp thanh toán trực tuyến VNPay.

**🌐 Live Demo:**
- Frontend: https://e-commerce-sigma-green.vercel.app
- Backend API: https://e-commerce-production-7be5.up.railway.app

## Chức năng

**Khách hàng:**
- Xem và tìm kiếm sản phẩm (Sách, CD/LP, DVD)
- Đăng ký và đăng nhập tài khoản
- Thêm sản phẩm vào giỏ hàng
- Đặt hàng và thanh toán qua VNPay
- Xem lịch sử đơn hàng và giao dịch

**Quản trị viên:**
- Quản lý sản phẩm (thêm, sửa, xóa)
- Quản lý đơn hàng
- Xem báo cáo thanh toán

## Công nghệ

**Frontend:** React + TypeScript + Vite + TailwindCSS  
**Backend:** Spring Boot 3.4.4 + Java 21 + Spring Security + JWT  
**Database:** MySQL 9.4  
**Payment:** VNPay Integration  
**Deployment:** Vercel + Railway
